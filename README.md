# 动态表单

## 调用方式
```ts
<app-dynamic-form [formConfig]="formConfig" [formValue]="formValue" [isEdit]="isEdit"></app-dynamic-form>
```
- formConfig: 表单配置
- formValue: 表单值
- isEdit: 是否可编辑

## 支持的control类型
- TEXT
- PASSWORD
- EMAIL
- NUMBER
- SEARCH
- TEL
- URL
- TEXTAREA
- CHECKBOX
- CHECKBOX_GROUP
- SELECT
- RADIO
- ADDRESS
- IMAGE
- IMAGES
- DATE
- YEAR
- MONTH
- WEEK
- RANGE
- DATA_ARRAY
